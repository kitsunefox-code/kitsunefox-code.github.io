# 草野球ユニフォーム比較ナビ

草野球チーム向けに「オーダーユニフォームをどこで作るか」を比較できる、SEO重視の静的サイト（Next.js）です。
広告（Google AdSense）とアフィリエイト／メーカー提携で収益化することを想定しています。

## 開発

```bash
npm install
npm run dev      # http://localhost:3000
```

## 本番ビルド（静的書き出し）

```bash
npm run build    # out/ に静的HTMLが出力されます
```

`out/` をそのまま Vercel / Netlify / GitHub Pages 等にアップロードすれば公開できます。
（Vercel なら GitHub 連携でプッシュ→自動デプロイが最も簡単です）

## ページ構成

| URL | 内容 |
|-----|------|
| `/` | おすすめランキング13選（mybest風・星評価/項目別スコア）＋全社一覧比較表 |
| `/shindan/` | ぴったり診断（イメージ写真の配色解析＋質問→おすすめ3社を提案） |
| `/guide/uniform-how-to/` | 作り方ガイド記事（SEO流入用） |
| `/about/` | 運営者情報・評価基準（AdSense審査・E-E-A-T対策） |
| `/privacy/` | プライバシーポリシー・免責事項（AdSense審査に必須） |

掲載13社：Fungo / ILB-MAX / WAKKUON / ユニフォームラボ / SORK / ZETT / SSK / UNIO / REWARD / ミズノ / イウジン / BWOS / アシックス
（デサントは2025年末で野球ブランド展開終了のため非掲載）

ランキングの星評価は `data/makers.ts` の `ratings`（5軸×5点満点）から自動計算されます。
配列の並び順＝総合評価順なので、順位を変えたいときは配列を並べ替えてください。

診断の写真解析はすべてブラウザ内（Canvas）で完結し、画像はどこにも送信されません。

## 収益化のための設定手順

**収益化まわりの設定は `data/site.ts` に集約されています。**

### 1. 独自ドメインを用意する
AdSense 審査・SEO のため独自ドメイン推奨。取得後、`data/site.ts` の `SITE_URL` を差し替えます（1ヶ所だけでOK）。

### 2. アフィリエイトリンクを入れる
`data/makers.ts` の各メーカーの `affiliateUrl` にリンクを設定すると、
比較表・診断結果の「公式サイトを見る」ボタンが自動でそのリンクに切り替わります（未設定なら公式URLのまま）。

- **A8.net / もしもアフィリエイト / バリューコマース**：登録（無料・要サイト審査）後、「ユニフォーム」「スポーツ」で広告主を検索して提携申請
- **楽天／Amazonアソシエイト**：野球用品（帽子・アンダー・小物）の関連商品に
- **メーカー直提携**：PV が育ってから各社に掲載料・成果報酬を交渉

### 3. Google AdSense を設置する
審査通過後、`data/site.ts` の `ADSENSE_CLIENT_ID` に `ca-pub-...` を設定するだけで、
全ページの広告枠（プレースホルダー）が実際の広告に切り替わります。
広告ユニットごとのスロットIDは同ファイルの `ADSENSE_SLOTS` に設定できます（未設定なら自動フォーマット）。

### 4. Google Analytics 4 を設置する
`data/site.ts` の `GA4_MEASUREMENT_ID` に測定ID（`G-...`）を設定すると全ページに解析タグが入ります。
アクセス実績はASP・メーカー直提携の交渉材料になるため、公開と同時の設置を推奨します。

### 5. Google Search Console に登録
`sitemap.xml`（自動生成・全5ページ登録済み）を Search Console に登録し、インデックスを促進します。

### AdSense 審査対策（実装済み）
プライバシーポリシー（/privacy/）・運営者情報（/about/）・お問い合わせ先（`data/site.ts` の `CONTACT_URL`）は
審査の必須要件のため実装済みです。`CONTACT_URL` は Google フォーム等に差し替え可能です。

### 法令対応（設定不要・実装済み）
- 全ページ最上部にPR表記（ステマ規制/景表法対応）
- アフィリエイトリンクに `rel="sponsored"` 付与
- 価格が「目安」である旨の免責表示

## データ更新

掲載メーカー・価格・仕様は `data/makers.ts` を編集するだけで反映されます。
価格は変動するため、定期的に各社公式サイトで確認して更新してください。

## 注意

掲載価格・納期は各社公開情報をもとにした目安です。景品表示法・各アフィリエイトプログラムの規約に沿って、
「PR表記」「価格は目安である旨」を明示しています（フッター・免責に記載済み）。
